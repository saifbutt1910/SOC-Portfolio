# Week 1 Day 2 - DNS and Web Basics

## What is DNS?
DNS (Domain Name System) converts website names into IP addresses so computers can find websites on the internet.

Example:
google.com -> IP address

## What are Packets?
Packets are small pieces of data sent across a network. Large data is broken into packets before transmission.

## What I Learned Today
- DNS helps users access websites easily without remembering IP addresses.
- Data travels through networks in packets.
- Websites communicate using requests and responses.
- Networking fundamentals are essential in cybersecurity.
