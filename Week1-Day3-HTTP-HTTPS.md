Week 1 Day 3 - HTTP and HTTPS

What is HTTP?
HTTP (HyperText Transfer Protocol) is used for communication between a browser and a website.

What is HTTPS?
HTTPS is the secure version of HTTP. It encrypts data so attackers cannot easily read sensitive information.

Why HTTPS is Important
HTTPS protects:
- passwords
- personal information
- banking data
- messages

What I Learned Today
- Websites use protocols to communicate.
- HTTPS is more secure than HTTP because it uses encryption.
- Cybersecurity depends heavily on secure communication.
